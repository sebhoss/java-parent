h1. Java Parent

p. Maven parent POM for Java-based projects. Defines several aspects of a build-process, such as

* License header
* Outputs -sources.jar
* Outputs -javadoc.jar
* Compiler settings
* OSGi infos
* Findbugs settings
* PMD settings
* Cobertura settings
