h1. Java Parent "!https://secure.travis-ci.org/sebhoss/java-parent.png!":http://travis-ci.org/sebhoss/java-parent

p. Maven parent POM for Java-based projects. Defines several aspects of a build-process, such as

* Outputs -sources.jar
* Outputs -javadoc.jar
* Compiler settings
* OSGi infos
* Findbugs settings
* PMD settings
* Cobertura settings
* Eclipse settings
* Outputs DOAP file
* Dependency analysis
* Dependency version check
* Duplicate check

p. Use it as follows:

bc. <parent>
  <groupId>com.github.sebhoss</groupId>
  <artifactId>java-parent</artifactId>
  <version>${version.java-parent}</version>
</parent>
